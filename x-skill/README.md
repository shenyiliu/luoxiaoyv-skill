# X Operations Skill

自动化运营 X (Twitter) 的 AI 助手。

## 概述

本 Skill 为 **OpenClaw 场景**设计，用于自动化运营 X 账号：

- **内容抓取**：使用 x-reader 抓取任意账号的推文内容
- **内容策划**：生成互动回复和原创推文草稿
- **每日运营**：自动化执行每日互动工作流

## 核心功能

### 1. 推文抓取
```bash
x-reader https://x.com/username
```
无需 API Key 获取目标账号的最新推文。

### 2. 每日运营工作流
- 从目标账号池选取互动对象
- 抓取最新推文
- 生成英文回复 + 中文参考
- 附带原创推文草稿
- 发送到 Telegram 等待用户手动发布

## 适用场景

- **账号运营**：需要批量管理 X 账号的个人或团队
- **内容策划**：生成互动回复和原创推文草稿
- **定时任务**：每日自动执行运营工作流

## 环境要求

- x-reader（推文抓取）
- Telegram（用于接收每日运营内容）

详细安装步骤见 [SKILL.md](./SKILL.md)

## 工作原理

```
用户 → Skill → x-reader 抓取 →  生成内容草稿 → Telegram 发送 → 用户确认 → 手动发布
```

本 Skill 专注于内容策划和生成，实际发布操作由用户手动完成。
